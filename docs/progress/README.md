# Progress Log

This folder is the append-only iteration log for the project.

After every iteration, create a new file here with this naming format:

- `YYYY-MM-DD_HHMM_phase-##_iter-##.md`

Use the template in:

- [`../Plans/_templates/PROGRESS_ENTRY_TEMPLATE.md`](../Plans/_templates/PROGRESS_ENTRY_TEMPLATE.md)

Rules:

- One file per iteration
- Do not overwrite old entries
- Include commands, evidence, results, blockers, and the next action
- Write the entry even if the iteration failed
