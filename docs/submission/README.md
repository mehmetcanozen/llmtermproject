# Submission Folder Index

This folder contains the final hand-in support materials for the LLM course project.

## Files

- `SUBMISSION_GUIDE.md`: video plan, demo checklist, narration script, and final submission checklist.
- `final_report_ieee/`: two-column IEEE-style final report package.

## Recommended Final Report File

Use `final_report_ieee/final_report.tex` as the canonical report source.

If you need a no-LaTeX path, open `final_report_ieee/final_report_print.html` in a browser and print it to PDF.

## Recommended Video Flow

Use `SUBMISSION_GUIDE.md`. It is organized around the required video topics:

- project goal,
- data or task description,
- LLM-based method,
- experiments and results,
- limitations,
- demo of the system and artifacts.
